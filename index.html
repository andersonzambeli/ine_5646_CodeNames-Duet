Descrição do Projeto

O projeto web (escrita, implementação e apresentação) deverá ser organizado em grupo de dois (2) discentes. Este projeto envolverá os principais conceitos e técnicas abordados na disciplina. A parte escrita do projeto será em formato de artigo em LATEX, conforme template disponibilizado. A apresentação (website HTML/CSS/JS/framework online) deverá contemplar aspectos téoricos (de forma sucinta) e focar principalmente em aspectos de implementação (aplicação) e de monstração/resultados. O agente de usuário padrão adotado na disciplina será o Firefox. Entretanto, recomenda-se que seja verificada a compatibilidade das implementações no Chrome.

O Jogo Codename: Duets

Codenames Duet mantém os elementos básicos do Codenames - dê pistas de uma palavra para tentar conseguir que alguém identifique seus agentes entre os que estão na mesa - mas agora você está trabalhando em equipe para encontrar todos os seus agentes. (Por que você ainda não sabe quem são os seus agentes é uma pergunta que os investigadores do Congresso terão em suas costas mais tarde!)

Regras do jogo

Colocam-se 25 palavras na mesa num gradil 5 x 5. Separam-se os marcadores de tempo de acordo com a missão escolhida. Os jogadores posicionam uma carta-chave no apoio, separam os marcadores de agentes e assassino e a partida está pronta para começar.
Cada jogador tem acesso a um lado da carta-chave, que indica quais palavras o outro jogador deve adivinhar, e para isso o jogador vai dar pistas para o outro usando apenas UMA palavra e UM número, sendo que esse número corresponde à quantidade de palavras que o outro jogador consegue adivinhar com essa pista (Por exemplo, se eu disser: Fruta 2, significa que com a pista fruta o outro jogador consegue adivinhar até 2 palavras).
Existem três possibilidades de resultado quando um jogador escolhe uma palavra: Descobrir um agente (marcações verdes na carta-chave), acertar um civil (marcação cinza) e acertar o assassino (marcação preta). Ao descobrir um agente você obteve sucesso. Ao acertar um civil o turno se encerra mas a partida prossegue, sendo que agora o outro jogador que fornecerá pistas para você, pois a carta-chave possui informações de ambos os lados, sendo que cada lado fica visível para apenas um jogador. Já ao acertar o assassino a partida encerra-se imediatamente e vocês perdem, sendo assim o assassino deve ser evitado à todo custo.
Essa versão do jogo vem com marcadores de tempo e civis para ser usado, pois cada missão do mapa possui uma quantidade de turnos permitidos para os jogadores descobrirem todas as palavras e também quantidade de erros permitidos.
Ao exceder o número de erros permitidos cada erro adicional custará 2 turnos aos jogadores.
A partida é vencida quando ambos os lados descobrirem todas as palavras indicadas pela carta-chave antes dos turnos indicados na missão terminarem.
